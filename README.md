# samp-gamemode
Simple samp party gamemode with dm arena, login/register system made in dfile and some other simple functions
not open source for now

required .inc files for pawno compiler:
include <a_samp>
include <dfile>
include <kolory>
include <Sscanf2>
include <zcmd>
include <streamer>

plugins:
whirlpool.dll
streamer.dll
sscanf.dll
